# Datasets disponibles

Los notebooks de las unidades 1-5 utilizan corpus incorporados en `quanteda` (por ejemplo, `data_corpus_inaugural`) y ejemplos de texto autocontenidos, por lo que no requieren datasets externos.

Los datasets para el proyecto integrador y las unidades de análisis de sentimiento (6) y similitud semántica (7) se publicarán aquí a medida que estén disponibles.
